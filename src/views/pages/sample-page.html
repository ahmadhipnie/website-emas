MVC Architecture

This website uses a clean MVC (Model-View-Controller) architecture:

  • Models - Data and business logic (src/models/)
  • Views - UI templates and components (src/views/)
  • Controllers - Handle user input and updates (src/controllers/)

Additional folders:

  • config - Configuration and routes
  • services - API and storage services
  • utils - Helper functions and event bus
